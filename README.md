# 🐍 Python Data Structures Assignment

## 📖 About This Project

This repository contains my Python Data Structures assignment, which was given by **Unified Mentor** as part of my learning program.

In this assignment, I worked on the basic data structures in Python like lists, strings, sets, tuples, and dictionaries. The main aim was to understand how these data structures work and how they are used in real coding.

---

## 🎯 What I Learned

Through this assignment, I was able to:

* Understand how to create and use different data structures
* Practice basic operations like adding, removing, and updating data
* Use loops and conditions with data structures
* Improve my basic Python programming skills

---

## 🛠️ Topics Covered

### Lists

I learned how to create lists, access elements, use loops, and perform operations like append and remove.

### Strings

I worked with string operations like splitting, converting to lowercase, joining words, and checking values.

### Sets

I understood how sets remove duplicates and how to perform operations like union, intersection, and difference.

### Tuples

I learned about immutable data and how tuples are different from lists.

### Dictionaries

I practiced working with key-value pairs, updating values, and even counting characters using dictionaries.
